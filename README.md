Simple opinionated dotfiles.

## Installation

On a new setup:

    git clone https://github.com/abdelq/dotfiles ~/.config

## Dependencies

- `dunst` : `firefox`, `rofi`
- `git` : `neovim`
- `neovim` : `fzf`, `ripgrep`, `vim-plug`, `words`
